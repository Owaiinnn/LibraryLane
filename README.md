# LibraryLane

Een digitale straatbibliotheek gebouwd in React. Gebruikers kunnen boeken bekijken, meenemen, teruggeven en doneren via de OpenLibrary API.

## Opstarten

```bash
npm install
npm run dev
```

---

## De opdracht

Projectomschrijving – Straatbibliotheek

De Straatbibliotheek is een frontend-project gebouwd in React waarin gebruikers digitaal boeken kunnen ruilen via een eenvoudige webinterface. Het project maakt gebruik van de OpenLibrary.org API om boekinformatie op te halen.

De applicatie start met een verzameling boeken waarvan enkel de OpenLibrary ID's lokaal zijn opgeslagen. Gebruikers kunnen:

* 📖 Boeken bekijken: Op de detailpagina wordt extra informatie opgehaald via de API.
* ➖ Boeken meenemen: Het boek verdwijnt uit de lokale lijst.
* ➕ Boeken toevoegen: Via een zoekfunctie voert de gebruiker een term in, selecteert een boek uit de resultaten, en de bijbehorende ID wordt opgeslagen.

De homepagina toont een overzicht van alle beschikbare boeken.

Er is ruimte voor extra features zoals favorieten, categorieën, beschikbaarheidsstatussen of een logboek van toegevoegde/verwijderde boeken.

De focus ligt op een duidelijke gebruikerservaring, eenvoudige API-integratie en het uitbreiden van React-vaardigheden in een praktisch, realistisch project.
